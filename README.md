# REST API MONGODB, EXPRESS JS, NODE JS
## Download 
Git Clone
```
git clone https://github.com/alirfanyasin/Rest-api-MEN.git
```

Format ZIP
```
https://github.com/alirfanyasin/Rest-api-MEN/archive/refs/heads/master.zip
```

## Usage
Directory
```
cd Rest-api-MEN
```
or
```
cd Rest-api-MEN-master
```

Install Package
```
npm install
```

Run Project
```
npm run dev
```
